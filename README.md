# Multilingual Support System

## Project Overview
Automated multilingual support request triage system for support Foundation NGO.

## Features
- Multilingual form (English, Hindi, Tamil, Telugu, Bengali, Marathi)
- Auto-categorizes requests: Medical, Financial, General
- AI-powered responses via Google Gemini
- Telegram alerts for urgent medical cases
- Google Sheets logging with error tracking

## Tools Used
- n8n (workflow automation)
- Google Gemini AI
- Gmail
- Telegram
- Google Sheets
- GitHub Pages (form hosting)
